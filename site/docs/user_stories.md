# User Stories – Learning Platform Backend

## 1. User Authentication & Roles

- ✅ As a visitor, I can register with name, email, and password so I can create an account.
- ✅ As a user, I can log in and receive a token so I can securely access my account.
- ✅ As a user, I can update my profile info (name, email, avatar) so my account reflects my identity.
- ✅ As a user, I can be assigned a role (‘registered’ or ‘admin’) so that my permissions are enforced.
- ✅ As a user, I can log out and revoke my token to end my session securely.

---

## 2. Content Browsing & Discovery

- ✅ As any user, I can list all published learning tracks and resources so I can explore available content.
- ✅ As any user, I can search content by title, tags, or type to find what matches my interest.
- ✅ As any user, I can filter content by difficulty, type, or skill to narrow my results.
- ✅ As any user, I can view the detail page for a resource or track to learn more about it.
- ✅ As any user, I can share content via a public link so others can access it directly.

---

## 3. Enrollment & Progress Tracking

- ✅ As a registered user, I can enroll in a learning track or resource so I can track my learning.
- ✅ As a registered user, I can view a list of my enrolled items in “My Learnings” to manage my learning path.
- ✅ As a registered user, I can mark an enrolled item as completed to record my progress.
- ✅ As a registered user, I can view completed items in my “Accomplishments” section.
- ✅ As a registered user, I can unenroll from an item to stop tracking it.

---

## 4. Content Creation & Management

- ✅ As a registered user, I can create a new draft learning track or resource so I can start contributing.
- ✅ As a registered user, I can edit my draft content to refine it.
- ✅ As a registered user, I can submit draft content for admin review to publish it.
- ✅ As a registered user, I can request deletion of my published content for removal from the platform.
- ✅ As a registered user, I receive notifications when my content is approved, rejected, or deleted.
- ✅ As an admin, I can list all content pending review or delete requests so I can moderate submissions.
- ✅ As an admin, I can approve or reject submitted content with optional feedback.
- ✅ As an admin, I can approve or reject deletion requests and optionally comment.
- ✅ As an admin, I can request changes from the content creator for quality reasons.
- ✅ As an admin, I can publish my own content directly without needing approval.

---

## 5. Review System

- ✅ As a registered user, I can leave a rating and comment on content I created or enrolled in.
- ✅ As a user, I can view ratings and comments on a resource or track to assess its value.
- ✅ As an admin, I can moderate or remove inappropriate reviews if needed.

---

## 6. Notifications

- ✅ As a user, I receive in-app and email notifications when relevant events occur (e.g. content approved, reviewed).
- ✅ As a user, I can view and mark notifications as read to manage them.

---

## 7. Skills & Tagging

- ✅ As an admin or content creator, I can associate skills with resources or tracks to define learning outcomes.
- ✅ As a user, I can filter or search content by skill to find aligned materials.

---

## 8. Content State Tracking

- ✅ As the system, I can store and retrieve state change history for all tracks and resources.
- ✅ As an admin or creator, I can view the change history for a given content item to understand its lifecycle.
